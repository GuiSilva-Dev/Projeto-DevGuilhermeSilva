## 💻 Sobre o Projeto

Este é um site de portfólio pessoal desenvolvido para apresentar minha jornada como **Desenvolvedor de Software** e estudante de Ciência da Computação. O projeto reúne meus principais trabalhos, habilidades técnicas e formas de contato, com foco em uma experiência de usuário interativa e visualmente moderna.

O design utiliza uma estética "Dark/Neon" com animações fluidas, fundo interativo e layout totalmente responsivo.

## ✨ Funcionalidades

*   **Animações Interativas:**
    *   Fundo com partículas interativas (`particles.js`).
    *   Efeito de digitação no texto de apresentação.
    *   Animações de entrada (scroll reveal) utilizando `IntersectionObserver`.
*   **Showcase de Habilidades:** Slider infinito (carrossel) automático para exibição de ícones de tecnologias (Soft Skills, Front-End, Back-End, Frameworks).
*   **Seção de Projetos:** Cards com efeitos de hover 3D e descrição detalhada dos projetos desenvolvidos.
*   **Formulário de Contato:** Integração funcional com **FormSubmit** para envio de e-mails diretamente pelo site.
*   **Responsividade:** Layout adaptável para Desktops, Tablets e Dispositivos Móveis.
*   **Menu Mobile:** Navegação otimizada com menu hambúrguer animado.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização avançada, Flexbox, Grid Layout, Animações (@keyframes) e Variáveis CSS.
*   **JavaScript (Vanilla):** Lógica de interatividade, manipulação do DOM e observadores de eventos.
*   **Bibliotecas/Recursos Externos:**
    *   Particles.js - Para o efeito de fundo.
    *   Bootstrap Icons - Ícones da interface.
    *   Google Fonts & FontAwesome.
